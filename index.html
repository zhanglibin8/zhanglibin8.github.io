继承

post @ 2016-12-14

继承: 子类继承父类的方法或者属性

1 .原型继承: 子类的原型指向父类的实例 B.prototype = new A();

  • 父类的私有和公有方法都是子类的公有方法
  • 无论是定义在父类实例上的还是原型上的方法，相对于子类来说都是公有方法

2.call继承 ： 在子类的构造函数中执行 C.call(this)

  • 父类.call(this)
  • 把父类的私有方法变成子类的私有方法

3.冒充对象继承 : 在子类的构造函数中遍历父类的一个实例。并且把所有遍历出的属性和方法，分别添加给子类实例。

  • 如果在遍历过程中不使用hasOwnProperty等方法过滤出公有属性，那么就是把父类的私有和公有方法都变成子类的私有方法,都是添加到子类实例上的方法

4．组合继承 原型继承+call继承

－　由于原型继承是把私有+公有全部都变成了子类的公有,call继承就是把私有变成私有. 组合在一起就子类的公有方法中多了一套父类的私有方法。但是调用时优先使用私有方法.即使公有上还有一个相同的，那么也不会出现问题

5.中间件继承 proto这个属性在低版本ie中是不可以使用的。

  • 子类的原型上proto属性本来应该是指向Object.prototype的。但是现在却指向父类的原型。所以就可以继承父类原型上的方法。但是由于父类的原型也存在proto属性，所以并不会破坏原来的只想。只不过是多了一个中间项。
  • 把父类的公有方法变成子类的公有方法

6.Object.xxx Object.prototype.xxx 不兼容

  • Object.create(obj); 作用: Object.create本身是一个方法，需要一个对象(obj)参数。 返回一个类的实例，并且这个类的原型是传进来的obj参数

数组

post @ 2016-12-14

数组的操作

常用方法

  • push

  • 向数组末尾增加项

  • 返回数组新长度

  • 直接改变原数组

  • 其他实现方法：ary[ary.length]=1

  • pop

  • 删除数组最后一项

  • 返回被删除项

  • 直接改变原数组

  • 其他实现方法：ary.length–

  • shift

  • 删除数组第一项

  • 返回被删除的那一项

  • 直接改变原数组

  • unshift

  • 向数组开头增加项

  • 返回数组新长度

  • 原数组改变

  • splice

  • splice(n,m)从索引n开始，删除m个元素

  • splice(n)从索引n开始，删除到数组末尾

  • ary.splice(ary.length-1,0)==ary.pop()

  • splice(n,m,x)从索引n开始删除m个，用x替换删除项

  • splice(n,0,x)从索引n开始，删除0个(不删除)，把x增加到n的前面

  • ary.splice(ary.length,0,x)==ary.push(x)

  • slice

  • slice(n,m)从索引n开始截取到m-1处

  • slice(0) 相当于赋值数组

  • concat 拼接

  • toString 转化为字符串

-join(‘,’) 按照固定符号把数组拆分

  • reverse 倒序数组

  • sort 排序数组(10以内)

  • sort(function(a,b){ return a-b })有小到大排序

  • sort(function(a,b){ return b-a })有大到小排序

  • forEach 遍历数组

更好用的方法

例： var arr=[{name:1},{name:2},{name:3},{name:4}];

  • 注意：还需要处理低版本IE的朋友请忽略以下方法

  • 增 push

  • 直接改变原数组

  • 删(过滤) filter

  • 返回的新数组才是过滤好的结果，原数组不变

  • 改 map

  • 返回的新数组才是改变后的结果，原数组不变

  • 查 find

  • 返回结果才是要查找的项，原数组不变
